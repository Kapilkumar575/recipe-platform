## 🚀 Project Overview

This is a full-stack Recipe Platform where users can explore, share, and manage recipes.
It demonstrates modern web development using React, Node.js, and MongoDB.

---

## 🛠 Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

---

## ✨ Features

* 🔐 User Authentication (Login/Register)
* 📄 Add & View Recipes
* 🔍 Search & Filter Recipes
* 👤 User Profile Page
* 📸 Image Upload
* ❤️ Like / Save Recipes

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/Kapilkumar575/recipe-platform.git
cd recipe-platform
```

### 2️⃣ Setup Backend

```
cd backend
npm install
npm run dev
```

### 3️⃣ Setup Frontend

```
cd frontend
npm install
npm run dev
```

---

## 🌐 Environment Variables

Create `.env` file in frontend:

```
VITE_API_URL=http://localhost:5000/api
```

Create `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
```
